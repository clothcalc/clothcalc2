Carton Calculation

CBM Calculation from CM

Formula:
CBM = (length * width * height * Qty) / 1000000 Explanation:
* The CBM (Cubic Meter) is calculated by multiplying the length, width, height, and carton quantity.
* The result is divided by 1000000 to convert from cubic centimeters to cubic meters.

CBF Calculation from Inch

Formula:
CBF = (length * width * height * Qty) / 1728
Explanation:
* The CBF (Cubic Foot) is calculated by multiplying the length in inches, width in inches, height in inches, and carton quantity.
* The result is divided by 1728 to convert from cubic inches to cubic feet.

Carton Consumption

Consumption Formula with CM Measurement:
(length + width + 6) * (width + height + 4) * 2 / 10000

Consumption Formula with inch Measurement:
(length + width + 2) * (width + height + 2) * 2 / 1550

Cost Formula:
Total Square meter * Quantity * Price per Square Meter

NB: Mesurement in CM, 6/2 and 4/2 is here allownce.
