Consumption calculation for T-shirt, Polo Shirt, Hoody, Sweat Shirt, Fleece top item, Knit Jacket etc

Consumption Without Armhole

Consumption Formula Without Armhole

Consumtion Formula in CM:
(Body length + Sleeve Length + 10) X (Half Chest + 4) X 2 X Fabric Weight / 10000000

Consumtion Formula in Inch:
(Body length + Sleeve Length + 10) X (Half Chest + 4) X 2 X Fabric Weight / 1550000

NB: 10 and 4 is here allownce.

Consumption With Armhole

Consumption Formula With Armhole

Consumtion Formula in CM:
(Body Length + 6 X Chest + 4) + (Sleeve Length + 6 X (Half Armhole X 2)) X 2 X Fabric Weight / 10000000

Consumtion Formula in Inch:
(Body Length + Allowance X Chest + Allownce) + (Sleeve Length + Allowance X (Half Armhole X 2)) X 2 X Fabric Weight / 1550000

NB: 6 and 4 is here allownce.

NB: If armhole not find in measurement sheet you can calculate armhole from below calculator.

Comment: Consumption with Armhole Measurement is more Acurate.

T-shirt/ Sweat shirt Neck Rib Consumption Calculator

Armhole Calculation

Formula for Inch
ArmHole = (Half Chest inch / 2) - 1.5inch;

Formula for Cm
ArmHole = (Half Chest cm / 2) - 3.81cm;



Polo Collar Length Calculation

(If you don't have full collar length or Collar circumstance)

Formula
Collar Length = (neck width + front neck drop) * 3 / 2;



Polo Collar Consumption Calculation

Formula
Collar Length * (collar height + 2) * gsm / 10000000




N.B: Flat knit collar is single layer fabric. So, no need to multiply by 2. If the collar rib is double then need multiply the collar length with 2 first. In the formula 2 is for allowance.

Hood Consumption

Consumption Formula

Consumtion Formula in CM:
(Hood Height + Allowance) x (1/2 Hood Width + Allowance) x 2 x GSM / 100,00,000

Consumtion Formula in Inch:
(Hood Height + Allowance) x (1/2 Hood Width + Allowance) x 2 x GSM / 1550000
